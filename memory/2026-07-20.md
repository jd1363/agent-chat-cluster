## ⚠️ claude code headless 调用的坑（重要）
1. `--permission-mode acceptEdits` 只放行文件编辑，**不放行 Bash** → 卡审批。要全自动用 `bypassPermissions`
2. slash command `/opsx:propose` 在 `-p` headless 下会 Skill 中断，改用**直白 prompt 描述 openspec CLI 三步流程**更稳
3. **写文件乱码**：claude 输出 UTF-8 但落盘按 GBK → 破折号等非 ASCII 变乱码。解法：prompt 里强制"用 ASCII/英文、普通连字符-"
